## **Setup**

- Install dependencies : npm install
- Start your client : npm start
- visit "http://localhost:3000/" in your browser

## **Project Stack**

 - React
 - Material UI 
 - Charts.js 
 - Axios
 - React-chartjs-2
 - React-countup
 - React-dom
